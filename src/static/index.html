Spaldellino

Italian Card Game

How to Play

  • 2-8 players using Italian deck (40 cards)
  • Each player starts with 5 lives
  • Phases: 2, 3, 4, 5, 4, 3 cards (repeating)
  • Guess how many turns you'll win each phase
  • Wrong guess = lose 1 life
  • Last player standing wins!
